### 说明

以下是两个可以转跳 URL 的 HTML 实例

## HTML

```
<html>
<head>
<meta http-equiv="Content-Type" content="text/html; charset=gb2312">
<meta http-equiv="refresh" content="0;URL=Your Url">
</head>
</body>
</html>
```

## Js

```
<script>
location.href="You Url";
</script>
```
